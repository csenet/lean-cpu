### CPUエミュレーターの簡単な使い方

[](./a.png)

- Code
プログラムを入力します
実行中は実行している行がハイライトされます

- Run
プログラムを実行します

- Run(step)
プログラムを1ステップづつ実行します

- Stop
プログラムを停止します

- Memory
実行時のメモリの状態を表示します

- acc
アキュムレータの状態を表示します

- idx
インデックスレジスタの状態を表示します
